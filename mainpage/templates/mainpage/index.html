Приложение SR-Sales-Forecast - прогнозирование продаж
на основе исторических данных с использованием моделей Prophet и AutoArima

Приложение выполняет прогноз продаж на следующие периоды на основе фактических продаж товаров. Для базового периода продаж можно выбрать периоды: 8 недель(56 дней), 10 недель (70 дней), 12 недель (84 дня). Прогноз продаж можно сделать на 1 неделю (7 дней), 2 недели (14 дней), 3 недели (21 день), 4 недели (28 дней). Для составления прогноза можно выбрать либо модель Prophet, либо модель AutoArima, которые используются для прогнозирования будущих значений на основе исторических временных рядов.

Основные разделы приложения:
Справочники по товарам

номенклатура товаров;

категории товаров;

поставщики товаров;

Прогноз продаж

выполняется группировка исторических продаж по категориям ;

задаются периоды для исторических данных (8, 10, 12 недель) и для прогноза продаж (1, 2, 3, 4 недели);

на основе заданного периода исторических продаж выполняется прогнозирование продаж по категориям с использованием моделей Prophet или AutoArima (Прогноз продаж по категориям);

для расчета прогноза продаж по отдельным товарам используется структура товаров в фактических продажах по категориям за заданный период исторических продаж (Прогноз продаж по товарам).

Данные по продажам

фактические продажи по номенклатуре товаров.

фактические продажи по категориям товаров;

фактические продажи по поставщикам товаров.
