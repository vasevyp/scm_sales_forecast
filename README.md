# О приложении

Приложение выполняет прогноз продаж на следующие периоды на основе фактических продаж товаров. Для составления прогноза можно выбрать либо модель Prophet, либо модель AutoArima, которые используются для прогнозирования будущих значений на основе исторических временных рядов.

## Основные разделы приложения:

• справочники (товары, категории товаров, поставщики товаров)

• продажи(по товарам, по категориям, по поставщикам)

• прогноз продаж (по категориям товаров, по товарам)

Данные справочников и продаж могут дополняться через загрузку CSV файлов или через форму на странице приложения.

Для работы с приложением необходимо зарегистрироваться.
